# prismagram

Instagram clone with Express + prisma + react + react-native

## User Stories

- [ ] Log in
- [ ] Like / Unlike a photh
- [ ] Comment on a photo
- [ ] Search by user
- [ ] Search by location
- [ ] See user profile
- [ ] Follow / Unfollow User
- [ ] Edit my profile
- [ ] Create account
- [ ] Upload photos
- [ ] Edit the photo (Delete)
- [ ] See the feed
